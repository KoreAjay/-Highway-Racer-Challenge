A fast-paced, top-down 2D racing game built entirely in vanilla HTML, CSS, and JavaScript. Challenge yourself to stay on the road in this winding, infinite highway-style track!

âœ¨ Features

Top-Down 2D Racing: Classic arcade racing action.

Highway Style: The car remains centered while the track scrolls around it, giving the feeling of a long, high-speed drive.

Responsive Controls: Seamless playability on both mobile devices (using dedicated touch buttons) and laptops/desktops (using keyboard input).

Realistic Drag: Experience reduced speed and heavy friction when driving off the road (onto the green area).

Performance Metrics: Real-time display of speed (in km/h) and elapsed race time.

Single-File Build: The entire game is contained in one index.html file, making it extremely easy to host and share.

Since this game is a single HTML file, no complex dependencies or build steps are required.

Local Run

Save the content of the index.html file.

Open the file directly in any web browser.

Hosting with GitHub Pages

This is the recommended way to share your game:

Create a new public GitHub repository (e.g., highway-racer-game).

Upload the index.html file to the root of the repository.

Go to the repository Settings > Pages.

Set the source to deploy from your main branch (main or master).

Wait a minute for GitHub to publish the site, and then share the resulting URL!
